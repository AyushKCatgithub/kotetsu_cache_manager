## 0.0.1

- Initial release of `kotetsu_cache_manager`.
- Added in-memory cache with LRU eviction.
- Added TTL-based expiry for cache entries.
- Added optional persistent storage using SharedPreferences.
- Added support for storing primitives, lists, maps, and custom models.
- Added serialization utilities with JsonSerializable interface.
- Added example application.
- Added comprehensive test suite for TTL, LRU, persistence, clearing, and concurrency.
